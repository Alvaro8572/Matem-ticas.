1.1 — Fundamentos

¿Qué son el Seno y el Coseno?

Las funciones trigonométricas más fundamentales explicadas desde cero.

📐

El Seno (sen)

El seno de un ángulo mide la posición vertical (altura) de un punto en el círculo unitario. En un triángulo rectángulo, es la razón entre el cateto opuesto y la hipotenusa.

Fórmula en triángulo rectángulo

sen(θ) = cateto opuesto / hipotenusa

En el círculo unitario

sen(θ) = coordenada Y del punto

📌 Características clave

  • Rango: siempre entre −1 y 1
  • Período: se repite cada 360°
  • Cero en: 0°, 180°, 360°
  • Máximo: 1 en 90°
  • Mínimo: −1 en 270°
📏

El Coseno (cos)

El coseno de un ángulo mide la posición horizontal (base) de un punto en el círculo unitario. En un triángulo rectángulo, es la razón entre el cateto adyacente y la hipotenusa.

Fórmula en triángulo rectángulo

cos(θ) = cateto adyacente / hipotenusa

En el círculo unitario

cos(θ) = coordenada X del punto

📌 Características clave

  • Rango: siempre entre −1 y 1
  • Período: se repite cada 360°
  • Cero en: 90°, 270°
  • Máximo: 1 en 0° y 360°
  • Mínimo: −1 en 180°

🎓 La intuition gráfica

En el círculo unitario: para cualquier ángulo θ, trazás un radio desde el centro. El punto donde toca la circunferencia tiene coordenadas:

  • x = cos(θ)
  • y = sen(θ)

En un triángulo rectángulo: con ángulo θ en un vértice:

  • sen(θ) = alto / incline
  • cos(θ) = ancho / incline

📐 sen²(θ) + cos²(θ) = 1

Esta es la identidad pitagórica — siempre se cumple, para cualquier ángulo.

1.3 — Trigonometría

Propiedades del seno y coseno

Las reglas fundamentales que se cumplen para todos los ángulos.

🔄

Período de 360°

Los valores se repiten exactamente cada vuelta completa.

sen(θ + 360°) = sen(θ)
📐

Identidad pitagórica

La suma de sus cuadrados siempre da 1. Viene de Pitágoras.

sen²θ + cos²θ = 1
🔢

Rango del seno

El seno nunca puede ser mayor que 1 ni menor que −1.

−1 ≤ sen(θ) ≤ 1
🔢

Rango del coseno

El coseno tampoco puede salir de ese rango.

−1 ≤ cos(θ) ≤ 1
🪞

Seno es función impar

Si cambias el signo del ángulo, el seno cambia de signo.

sen(−θ) = −sen(θ)
🪞

Coseno es función par

Cambiar el signo del ángulo no cambia el coseno.

cos(−θ) = cos(θ)
🤝

Relación entre sen y cos

Son la misma función, pero desplazadas 90° entre sí.

sen(θ) = cos(90° − θ)
➗

La tangente

La tangente es simplemente el cociente entre sen y cos.

tan(θ) = sen(θ) / cos(θ)
📍

Valores especiales (sen)

Valores clave a memorizar.

sen(0°)=0, sen(90°)=1, sen(180°)=0
📍

Valores especiales (cos)

El coseno empieza en 1 y baja.

cos(0°)=1, cos(90°)=0, cos(180°)=−1

2.1 — ¿Qué es una Potencia?

¿Qué es una Potencia?

La forma compacta de expresar multiplicaciones repetidas.

📓 La definición

Una potencia es una forma abreviada de escribir una multiplicación repetida del mismo número.

Definición formal

an = a × a × ... × a

(n veces)

  • a = base (el número que se multiplica)
  • n = exponente (cuántas veces)
  • an = potencia

📝 Ejemplo visual

2⁴ significa:

2 × 2 × 2 × 2 = 16

→ 2⁴ = 16

3³ significa:

3 × 3 × 3 = 27

→ 3³ = 27

🌟 Casos especiales importantes

a⁰ = 1

Cualquier base elevada a 0 da 1

5⁰ = 1, 100⁰ = 1

a¹ = a

Exponente 1 = la misma base

7¹ = 7, (−3)¹ = −3

a−n = 1/an

Exponente negativo = inverso

2−3 = 1/8

2.2 — Propiedades

Propiedades de las Potencias

Las reglas que siempre se cumplen al trabajar con potencias.

✖️

Producto de misma base

Al multiplicar potencias con la misma base, los exponentes se suman.

am · an = am+n
Ej: 2³ · 2⁴ = 2⁷ = 128
➗

Cociente de misma base

Al dividir potencias con la misma base, los exponentes se restan.

am / an = am−n
Ej: 2⁵ / 2² = 2³ = 8
🎯

Potencia de potencia

Cuando elevás una potencia a otra potencia, multiplicás los exponentes.

(am)n = am·n
Ej: (2³)² = 2⁶ = 64
📦

Producto de mismo exponente

Podés agrupar las bases primero, el exponente se mantiene.

an · bn = (a·b)n
Ej: 2³ · 3³ = (2·3)³ = 6³
🔄

Cociente de mismo exponente

Podés dividir las bases primero, el exponente se mantiene.

an / bn = (a/b)n
Ej: 6³ / 3³ = (6/3)³ = 2³
♻️

Exponente cero

Cualquier número (excepto 0) elevado a 0 es 1.

a⁰ = 1 (a ≠ 0)
Ej: 5⁰ = 1, (−7)⁰ = 1

3.1 — ¿Qué es una Raíz?

¿Qué es una Raíz?

La operación inversa a la potenciación.

📓 La definición

La raíz es la operación inversa a la potenciación. Si una potencia eleva un número a una potencia, la raíz encuentra la base.

Definición formal

√n√a = b ⟺ bn = a

Se lee: "la raíz enésima de a es igual a b".

📝 Ejemplo visual

√16 = 4 porque:

4² = 16

³√27 = 3 porque:

3³ = 27

√ sin número = raíz cuadrada (n=2)

🔢 Tipos de raíces

√a

Raíz cuadrada

n = 2

√16 = 4

³√a

Raíz cúbica

n = 3

³√8 = 2

⁴√a

Raíz cuarta

n = 4

⁴√81 = 3

ⁿ√a

Raíz enésima

n = cualquier

⁵√32 = 2

⚠️ Diferencia crucial: raíces pares vs impares

Raíces pares (√, ⁴√, ...)

  • Solo operan sobre números ≥ 0
  • Tienen 2 soluciones (±)
  • √4 = ±2

Raíces impares (³√, ⁵√, ...)

  • Operan sobre cualquier número
  • Tienen 1 solución única
  • ³√−8 = −2

3.2 — Propiedades

Propiedades de las Raíces

Las reglas que siempre se cumplen al trabajar con raíces.

✖️

Producto bajo raíz

La raíz de un producto es el producto de las raíces.

√n√(a·b) = √n√a · √n√b
Ej: √16 = √(4·4) = √4·√4 = 2·2
➗

Cociente bajo raíz

La raíz de un cociente es el cociente de las raíces.

√n√(a/b) = √n√a / √n√b
Ej: √(16/4) = √16/√4 = 4/2
🎯

Raíz de raíz

Podés "apilar" las raíces multiplicando los índices.

⁽ᵐ⁾√⁽ⁿ⁾√a = ⁽ᵐ·ⁿ⁾√a
Ej: ²√³√8 = ⁶√8 = 2
📦

Raíz como potencia

Cualquier raíz se puede escribir como potencia.

ⁿ√a = a1/n
Ej: √a = a1/2, ³√a = a1/3
🔄

Potencia bajo raíz

Elevá la raíz a una potencia para simplificar.

(ⁿ√a)m = ⁿ√am
Ej: (√8)² = ⁵√8² = 8
♻️

Raíz de exponente

Podés dividir exponente e índice.

ⁿ√am = ⁽ⁿ÷ᵐ⁾√am÷n
Ej: ⁶√8² = ³√8¹ = ³√8 = 2

4.1 — Definición

¿Qué es un Triángulo?

La figura geométrica más básica con 3 lados y 3 ángulos.

📐 La definición

Un triángulo es un polígono de 3 lados, 3 vértices y 3 ángulos. Es la figura geométrica más simple que existe después de la línea.

  • Tiene 3 lados (segmentos que unen vértices)
  • Tiene 3 vértices (puntos de intersección)
  • Tiene 3 ángulos (en el interior)
  • La suma de sus ángulos siempre es 180°

📝 Elementos

  • Vértices: A, B, C
  • Lados: a, b, c
  • Ángulos: α, β, γ

📏 Perímetro y Área

Perímetro

P = a + b + c

Área (Fórmula de Herón)

A = √s(s-a)(s-b)(s-c)

s = P/2 (semiperímetro)

Área (base × altura)

A = (b × h) / 2

4.2 — Clasificación

Clasificación de Triángulos

4.3 — Teoremas

Triángulo Rectángulo

Los teoremas fundamentales que cumplen los triángulos rectángulos.

📐 Teorema de Pitágoras

En todo triángulo rectángulo, el cuadrado de la hipotenusa es igual a la suma de los cuadrados de los catetos.

c² = a² + b²

Calcular hipotenusa

c = √(a² + b²)

Calcular cateto

a = √(c² - b²)

📏 Teorema de la Altura

En un triángulo rectángulo, la altura sobre la hipotenusa es media proporcional entre las proyecciones de los catetos.

h² = m · n

h = altura, m y n = proyecciones de los catetos

📐 Teorema de los Catetos

Cada cateto es media proporcional entre la hipotenusa y su proyección sobre ella.

Cateto a

a² = c · m

Cateto b

b² = c · n

📓 Teorema de la Mediana

En un triángulo rectángulo, la mediana a la hipotenusa es igual a la mitad de la hipotenusa.

m = c / 2

m = mediana a la hipotenusa

🔄 Teoremas Recíprocos

Recíproco de Pitágoras:

Si en un triángulo se cumple a² + b² = c², entonces es un triángulo rectángulo.

Aplicaciones:

  • Identificar triángulos rectángulos
  • Clasificar triángulos
  • Verificar si un triángulo es rectángulo

4.3b — Isósceles

Triángulo Isósceles

Un triángulo isósceles tiene dos lados iguales y dos ángulos iguales.

📐 Características

  • Dos lados iguales: los lados iguales se llaman "catetos" o "lados iguales"
  • Ángulos iguales: los ángulos opuestos a los lados iguales son iguales
  • Altura = Mediana: la altura, mediana y bisectriz desde el vértice superior coinciden
  • Eje de simetría: la línea que pasa por el vértice y el punto medio de la base

⚖️ Teoremas del Isósceles

h = √(a² - (b/2)²)

Altura en términos de los lados iguales (a) y la base (b)

A = (b × h) / 2

Área = base × altura / 2

α = β

Los ángulos en la base son iguales

4.4 — Cualesquiera

Teoremas en Triángulos Cualesquiera

Teoremas que aplican a cualquier tipo de triángulo.

⚖️ Teorema de la mediana (Apolonio)

En cualquier triángulo, la mediana une un vértice con el punto medio del lado opuesto.

AB² + AC² = 2(AM² + BM²)

donde M es el punto medio del lado BC

📐 Ley de los senos

En todo triángulo, la proporción entre cada lado y el seno de su ángulo opuesto es constante.

a/sen(α) = b/sen(β) = c/sen(γ)

📏 Ley de los cosenos

Generalización del teorema de Pitágoras para triángulos cualesquiera.

c² = a² + b² − 2ab·cos(γ)

Para γ obtuso: c² > a² + b²

Para γ agudo: c² < a² + b²

5.1 — ¿Qué es un Logaritmo?

¿Qué es un Logaritmo?

La definición completa que necesitás para entender logarithmos de verdad.

📓 La definición

Un logaritmo es la operación inversa a la exponenciación. Responde a la pregunta:

¿A qué potencia debo elevar la base para obtener este número?

loga(x) = y ⟺ ay = x

Se lee: "el logaritmo base a de x es igual a y".

📝 Ejemplo concreto

Si 2³ = 8, entonces:

log₂(8) = 3

  • Base: 2 (el número que elevamos)
  • Argumento: 8 (el resultado buscado)
  • Resultado: 3 (el exponente)

🤔 ¿Por qué existen los logaritmos?

Antes de las calculadoras: multiplicar números grandes era difícil. Los logaritmos convierten:

  • ❌ Multiplicación → ✅ Suma
  • ❌ División → ✅ Resta
  • ❌ Potenciación → ✅ Multiplicación

Usos reales: los logaritmos están en todos lados:

  • 🔊 Escala de Richter (terremotos)
  • 🔊 Decibeles (sonido)
  • 🧬 pH (Química)
  • 💰 Interés compuesto (Finanzas)

🔢 Bases más comunes

log₁₀

Base 10

"logaritmo común"

log₁₀(100) = 2

ln

Base e ≈ 2.718

"logaritmo natural"

ln(e) = 1

log₂

Base 2

"logaritmo binario"

log₂(8) = 3

⚠️ Condición indispensable

La base a debe ser mayor que 0 y diferente de 1 (a > 0, a ≠ 1). Además, el argumento x debe ser positivo (x > 0).

✓ a > 0 ✓ a ≠ 1 ✓ x > 0

5.2 — Propiedades

Propiedades de los logaritmos

Un logaritmo responde: ¿a qué potencia elevo la base para obtener ese número? Si 2³ = 8, entonces log₂(8) = 3.

Propiedad 01

📦 Producto → suma

log(A · B) = log(A) + log(B)

Multiplicar dentro del log se convierte en sumar los logaritmos.

log(2 · 8) = log(2) + log(8) = log(16)
Propiedad 02

✂️ Cociente → resta

log(A / B) = log(A) − log(B)

Dividir dentro del log se convierte en restar los logaritmos.

log(8 / 2) = log(8) − log(2) = log(4)
Propiedad 03

⬆️ Potencia → multiplica

log(Aⁿ) = n · log(A)

El exponente "baja" y se multiplica por delante del logaritmo.

log(8³) = 3 · log(8)
Propiedad 04

🎯 Logaritmo de 1

logₐ(1) = 0

Cualquier base elevada a 0 es 1, por eso el log de 1 siempre es cero.

log₁₀(1) = 0, log₂(1) = 0
Propiedad 05

🏠 Logaritmo de la base

logₐ(a) = 1

¿A qué potencia elevo 2 para obtener 2? A la primera.

log₂(2) = 1, log₁₀(10) = 1
Propiedad 06

🔄 Cambio de base

logₐ(b) = log(b) / log(a)

Convierte cualquier logaritmo a base 10 o base e para calcularlo fácil.

log₂(8) = log(8)/log(2) = 3
