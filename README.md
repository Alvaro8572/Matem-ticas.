# Matemática — Web Interactiva de Matemáticas

Sitio web educativo de matemáticas interactivas con animaciones en tiempo real, resúmenes y ejercicios.

## Contenido

### 1. Seno y Coseno
- 1.1 Fundamentos — Definición de seno y coseno
- 1.2 Círculo Trigonométrico — Visualizador interactivo con círculo y onda
- 1.3 Propiedades — Identidad pitagórica, rango, paridad
- 1.4 Tabla de Referencia — Tabla de ángulos especiales (0°-360°)
- **Resumen + Ejercicios (5)**

### 2. Potencias
- 2.1 ¿Qué es una Potencia? — Definición y casos especiales
- 2.2 Propiedades — Producto, cociente, potencia de potencia

### 3. Raíces
- 3.1 ¿Qué es una Raíz? — Definición y tipos
- 3.2 Propiedades — Producto, cociente, raíz de raíz

### 4. Triángulos
- 4.1 Definición — Elementos del triángulo
- 4.2 Clasificación — Por lados y por ángulos
- 4.3 Teoremas — Pitágoras, altura, catetos, medianas
- 4.3b Isósceles — Características y teoremas especiales
- 4.4 Teoremas Cualesquiera — Teorema de Pitágoras generalizado
- **Resumen + Ejercicios (5)**

### 5. Logaritmos
- 5.1 ¿Qué es un Logaritmo? — Definición
- 5.2 Propiedades — Producto, cociente, potencia
- **Resumen + Ejercicios (5)**

## Características

- **Menú árbol** — Navegación lateral con ramas expandibles
- **Buscador** — Busca secciones en el menú
- **Canvas interactivos** — Animacionesdrawn con JavaScript
- **Código separado** — JavaScript en archivo separado (script.js)
- **Diseño oscuro** — Tema visual moderno

## Archivos

```
/Matemática
├── index.html    # Página principal
├── script.js    # JavaScript (funciones, canvas, resúmenes)
└── README.md    # Este archivo
```

## Cómo usar

1. Abrir `index.html` en un navegador
2. Navegar con el menú lateral
3. Interactuar con los sliders y botones
4. Hacer los ejercicios al final de cada sección

## Tecnologías

- HTML5
- CSS3 (Custom properties, grid, flexbox)
- JavaScript (Canvas API, ES6+)
